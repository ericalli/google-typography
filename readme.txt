A simple plugin that lets you use and customize (in real-time!) any fonts from Google Fonts on your existing WordPress site.

==Description==

The Google Typography plugins allows you to use any [Google Font](http://www.google.com/fonts) for any text on your existing website without writing a single line of code. In the Appearance > Typography settings page, simply add and customize as many fonts as you'd like, then apply it to any existing HTML tag or CSS selector.

==Frequently Asked Questions==

= What is this plugin and why do I need it? =

Google Typography provides a way for you to easily use any of the 600+ fonts from [Google Fonts](http://www.google.com/fonts) on ANY WordPress theme without having to modifying any of your theme files. Simply add and customize as many fonts as you'd like in your WordPress admin panel, assign them to specific CSS selectors on your site and Google Typography will automatically add all the necessary stylesheets and CSS to your theme.

= Is the list of Google Fonts updated in real time? =

Yes, the list of Google Fonts you get to choose from is updated every time you load the plugin page so you can pick form the freshest fonts Google has to offer.

==Installation==

This plugin can be installed directly from your site:

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Google Typography" into the Search input and click the "Search Widgets" button.
3. Locate the Google Typography in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. Navigate to Appearance &rarr; Typography to follow the on-screen instructions to begin using Google Typography.

It can also be installed manually:

1. Download the "Google Typography" plugin from WordPress.org.
2. Unzip the package and move it to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Google Typography" in the list and click the "Activate" link.
5. Navigate to Appearance &rarr; Typography to follow the on-screen instructions to begin using Google Typography.
